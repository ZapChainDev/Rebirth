# Rebirth and New Hope

A minimalist black and white memecoin landing page built with Next.js.

## Message

This project represents a call to action: stop dwelling on market negativity and embrace current opportunities.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- React 19

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Design Philosophy

- Minimalist black and white aesthetic
- Clean typography
- Simple, powerful message
- No distractions

## Build

```bash
npm run build
npm start
```
